# vue-top-movies
A full-stack web application built with Vue2 and Flask. Get a glimpse of IMDB top 250 rated movies and simple analytics with web crawler.
## Built With
- Vue2
- Flask
- Antd
## Setup
Install dependencies:
```
yarn install
pipenv install 
```
## Launch
```
./client
yarn serve

./
piepenv run start
```
